問題: 1. 1から10までの整数が含まれる配列 `numbers` があります。この配列から偶数だけを取り出して新しい配列 `evenNumbers` を作成してください。 2. 新しい配列 `evenNumbers` の要素をすべて足し合わせた合計を計算し、その値をコンソールに出力してください。 この問題を解決するために、`for` ループや配列のメソッドなどを活用してください。また、各ステップのコードを分割して段階的に解決してみてください。

reduce メソッド

例えば、友達がアイスクリームの費用を集めています。彼らの名前と集めた金額を配列で表すことができます。 ```javascript const friends = [ { name: "太郎", amount: 500 }, { name: "花子", amount: 700 }, { name: "次郎", amount: 300 }, { name: "三郎", amount: 600 } ]; ``` これらの友達が集めた金額を合計するには、`reduce` メソッドを使用します。 ```javascript const totalAmount = friends.reduce((sum, friend) => sum + friend.amount, 0); ``` ここでの処理は次の通りです。 - `sum`: 現在の合計金額です。最初は0から始まります。 - `friend`: 現在の友達の情報です。名前と集めた金額が含まれています。 最初の処理では、`sum` は0で初期化されます。次に、各友達の金額が `sum` に加算されていきます。 例えば、太郎が500円、花子が700円、次郎が300円、三郎が600円を集めている場合、合計金額は次のように計算されます。 ``` 0 + 500 (太郎) = 500 500 + 700 (花子) = 1200 1200 + 300 (次郎) = 1500 1500 + 600 (三郎) = 2100 ``` したがって、最終的な合計金額は2100円となります。 この例では、`reduce` メソッドを使って友達が集めた金額を合計しています。このメソッドを理解するには、各ステップでの合計金額の変化を追跡することが重要です。
