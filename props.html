In the next sections, you will be introduced to some core concepts of React from a JavaScript perspective. Here's a summary of the JavaScript topics that will be mentioned: Functions and Arrow Functions Objects Arrays and array methods Destructuring Template literals Ternary Operators ES Modules and Import / Export Syntax While this course does not dive into JavaScript, it's good practice to stay up to date with the latest versions of JavaScript. Functionの基本: 名前が add で、2つの引数を受け取り、それらを加算して結果を返す関数を作成してください。 この関数を使用して、5と3を加算した結果を出力してください。 Functionの引数: 名前が greet で、1つの引数 name を受け取り、"Hello, {name}!" というメッセージを返す関数を作成してください。 この関数を使用して、自分の名前を引数として渡し、挨拶を表示してください。 Arrow Functionの基本: Arrow Functionを使用して、名前が multiply で、2つの引数を受け取り、それらを掛け算して結果を返す関数を作成してください。 この関数を使用して、10と4を掛けた結果を出力してください。

Arrow Functionの引数: Arrow Functionを使用して、名前が greetShort で、1つの引数 name を受け取り、"Hi, {name}!" というメッセージを返す関数を作成してください。 この関数を使用して、友人の名前を引数として渡し、挨拶を表示してください。 関数のデフォルト引数: 名前が calculateArea で、引数 radius と pi を受け取り、円の面積を計算して返す関数を作成してください。pi のデフォルト値は3.14としてください。 この関数を使用して、半径が5の円の面積を計算して表示してください。 無名関数とコールバック関数: 無名関数を使用して、配列 [1, 2, 3, 4, 5] の各要素に2を掛ける関数を作成してください。 map() メソッドを使用して、この関数を配列に適用し、新しい配列を作成してください。結果を出力してください。
