map() メソッド

分かりやすい例えとして、果物のリストを考えてみましょう。例えば、次のような果物のリストがあるとします。 ```javascript const fruits = ["apple", "banana", "orange", "strawberry"]; ``` この果物のリストに対して、各果物名の先頭に "delicious" を追加した新しいリストを作成したいとします。ここで `map` メソッドを使用すると、簡単にこの操作を行うことができます。 ```javascript const deliciousFruits = fruits.map((fruit) => { return "delicious " + fruit; }); console.log(deliciousFruits); // 出力: ["delicious apple", "delicious banana", "delicious orange", "delicious strawberry"] ``` この例では、`map` メソッドを使用して元の果物のリストから新しいリストを生成しました。各果物名に "delicious " を追加する操作が、各要素に適用され、新しいリストが作成されました。これにより、元のリストを変更せずに、各果物名を変換した新しいリストを簡単に作成することができます。

`map` メソッドのコールバック関数には、実際には2つのパラメータがありますが、そのうちの1つがインデックスではなく、`要素（element）` の値自体です。正しい構文は以下の通りです： ```javascript array.map((element, index, array) => { // コールバック関数の処理 }); ``` `element` は現在の要素の値を表し、`index` はその要素のインデックスを表します。また、`array` は `map` メソッドが呼び出された配列そのものを表します。ただし、`array` パラメータは一般的にはあまり使用されません。 したがって、`map` メソッドのコールバック関数には2つのパラメータがありますが、2番目のパラメータはインデックスであり、3番目のパラメータは配列そのものです。
